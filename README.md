# E-Commerce Growth Intelligence Dashboard

## Project Overview
This project analyzes e-commerce sales data to generate business insights related to sales performance, customer behavior, and product profitability.  
The project follows a complete data analytics pipeline including data cleaning, database modeling, and interactive dashboard creation.

The analysis helps identify:
- High value customer segments
- Most profitable product categories
- Sales and profit trends

---

## Tools & Technologies Used

- Python (Pandas, NumPy)
- MySQL
- SQL
- Power BI
- Data Visualization

---

## Data Pipeline

Raw Data → Python Data Cleaning → MySQL Database → Star Schema Modeling → Power BI Dashboard

Steps performed:

1. Data cleaning and preprocessing using Python.
2. Data stored in MySQL database.
3. Star schema data model created for analytics.
4. RFM analysis performed to segment customers.
5. Interactive dashboards created using Power BI.

---

## Dashboard Pages

### 1. Executive Overview
Provides a high level summary of business performance.

Includes:
- Total Sales
- Total Profit
- Profit Margin
- Monthly Sales Trend
- Category Profit Contribution
- Revenue by Customer Segment

---

### 2. Customer Intelligence
Customer segmentation analysis using RFM model.

Segments:
- Loyal Customers
- Champions
- Potential Customers
- At Risk Customers

Insights include:
- Customer distribution
- Revenue by segment
- Profit by segment
- Customer retention opportunities

---

### 3. Product & Profit Intelligence
Analyzes product performance and profitability.

Includes:
- Top performing sub-categories
- Profit by product category
- Sales vs Profit scatter analysis
- Product profitability insights

---

## Key Insights

- Loyal customers contribute the highest overall revenue.
- Technology category generates the highest profit.
- Phones and Accessories are top performing sub-categories.
- Furniture category shows lower profit margins despite strong sales.
